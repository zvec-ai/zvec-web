---
title: Grouped Query
extendedTitle: Grouped Query (Coming Soon)
---


Grouped Query brings the power of SQL-like `GROUP BY` semantics to vector search. With this feature, you'll be able to group retrieval results by specified scalar fields — such as `category` — and perform vector similarity searches within each group.

<Callout className="text-base" type="info">
    Grouped Query support will be available in an upcoming release.
</Callout>
