## Create a Collection

A [collection](../collections/) stores your documents. Each [document](../concepts/data-modeling/#documents) contains scalar fields and [vector embeddings](../concepts/vector-embedding/).

First, define a schema ‚Äî then create a collection:

<Accordions type="single">
  <Accordion title="Code">

  ```python title="Create a collection"
  from centaurdb import db
  from centaurdb.model.param import CollectionOption, HnswIndexParam, InvertIndexParam
  from centaurdb.model.schema import FieldSchema, VectorSchema
  from centaurdb.typing import DataType, MetricType

  collection_schema = db.CollectionSchema(    # [!code highlight]
      name="my_collection",
      fields=[
          FieldSchema(
              name="publish_year",
              data_type=DataType.INT32,
              index_param=InvertIndexParam(enable_range_optimization=True),
          ),
      ],
      vectors=[
          VectorSchema(
              name="embedding",
              data_type=DataType.VECTOR_FP32,
              dimension=768,
              index_param=HnswIndexParam(metric_type=MetricType.COSINE),
          ),
      ],
  )


  collection = db.create_and_open(            # [!code highlight]
      path="/path/to/my/collection",
      schema=collection_schema,
      option=CollectionOption(read_only=False, enable_mmap=True),
  )
  ```

  <Callout className="text-base" type="info">
    **Important**: The field names you define here (`publish_year`, `embedding`) must be used exactly as written when inserting or querying data.
  </Callout>

  </Accordion>
</Accordions>

## Add Documents

Insert documents with scalar fields and vector embeddings:

<Accordions type="single">
  <Accordion title="Code">

  ```python title="Insert a document"
  from centaurdb.model import Doc

  collection.insert(    # [!code highlight]
      Doc(
          id="book_1",
          vectors={  # ‚Üê Replace with a real 768-dim embedding in practice
              "embedding": [0.1 for _ in range(768)]
          },
          fields={
              "publish_year": 1936,
          },
      )
  )
  ```

  </Accordion>
</Accordions>

## Retrieve a Document by ID

Fetch a document directly by its `id`:

<Accordions type="single">
  <Accordion title="Code">

  ```python title="Fetch a document"
  result = collection.fetch(ids="book_1")   # [!code highlight]
  print(result)
  ```

  </Accordion>
</Accordions>

## Search with Vectors

### Basic Similarity Search

Find documents most similar to a given vector embedding:

<Accordions type="single">
  <Accordion title="Code">

  ```python title="Basic similarity search"
  from centaurdb.model import VectorQuery

  result = collection.query(    # [!code highlight]
      VectorQuery(
          field_name="embedding",
          vector=[0.1 for _ in range(768)],   # Replace with a real 768-dim embedding in practice
      ),
      topk=10,
  )
  print(result)
  ```

  </Accordion>
</Accordions>

Results are ranked by similarity score.

### Filtered Similarity Search

Combine vector search with conditional filters:

<Accordions type="single">
  <Accordion title="Code">

  ```python title="Filtered similarity search"
  from centaurdb.model import VectorQuery

  result = collection.query(          # [!code highlight]
      VectorQuery(
          field_name="embedding",
          vector=[0.1 for _ in range(768)],   # Replace with a real 768-dim embedding in practice
      ),
      topk=10,
      filter="publish_year > 1936",   # [!code highlight]
  )
  print(result)
  ```

  </Accordion>
</Accordions>

Only matching documents are considered during search.
